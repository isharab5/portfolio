# Senior AI/ML Engineer Portfolio

A modern, tech-forward portfolio website built with React, TypeScript, and Tailwind CSS.

## Pages

| Route | Description |
|-------|-------------|
| `/` | **Profile** — original portfolio (Hero, About, Experience, Skills, Research, Education, Contact) |
| `/agentic-ai` | **Agentic AI** — tabs for Multi-Agentic Workflow & LLM Fine-Tuning projects |
| `/machine-learning` | **Machine Learning** — sports analytics, time-series, computer vision research |
| `/full-stack` | **Full-Stack Development** — WSO2, New York Life, Axiata projects |
- Responsive design for all screen sizes
- Smooth scroll animations with Framer Motion
- Sections: Hero, About, Experience, Skills, Research, Education, Contact
- Glassmorphism UI with cyan/purple accent gradients

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customization

Edit `src/data/profile.ts` to update:
- Your name and contact links (LinkedIn, Google Scholar, email)
- Work experience, education, skills, awards, and research

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons
